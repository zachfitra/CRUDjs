tutorial fullstack CRUD react dengan nodejs
-buat folder project
- buat folder backend
-masuk ke folder backend dengan terminal
- install package json didalam folder backend dengan mengetikan "npm init -y"
-install dependensi yang dibutuhkan dengan mengetikan perintah " npm i express mysql2 sequalize cors"
-tambahkan "type" = "module" dibawah description

-buat folder config dalam folder backend lalu create file database.js
-buat folder model dalam folder backend

kode menggunakan react dengan mengetikan pada terminal "npx create-react-app ''nama folder'
-pada frontend depedensi yang dibutuhkan adalah react-router-dom,axios,bullma css


**
sebelum melanjutkan project yg harus di pastikan adalah
-server backend harus di nyalakan terlebih dahulu dengan cara memasukan perinntah pada terminal "nodemon index"
-lalu untuk menjalankan server react dengan mengetikan perintah" npm start"
